# ApplyFlow Session Capture Extension

This extension captures authenticated sessions from job portals to bypass login checks.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked** in the top left.
4. Select this `extension/` folder.

## Usage

1. Open a regular Chrome tab and log into LinkedIn, Naukri, Internshala, or Unstop.
2. While on that tab, click the ApplyFlow extension icon.
3. Click **Capture** for the corresponding platform.
4. The bot will now use this session instead of trying to log in automatically.
